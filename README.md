# Agrinho 2026 — Agro Forte, Futuro Sustentável 🌱 

Projeto web desenvolvido para o Concurso Agrinho 2026, focado na consciencialização sobre o equilíbrio entre a produção agrícola e a preservação ambiental no estado do Paraná.

## 🚀 Sobre o Projeto

O site aborda de forma clara e reflexiva os desafios da nossa geração para alimentar a população mundial sem destruir os recursos naturais. Destaca práticas reais do homem do campo, como o Plantio Direto, a Agricultura de Precisão e a proteção das Matas Ciliares, provando que a tecnologia e a ecologia caminham juntas.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído de raiz, utilizando tecnologias web nativas para garantir leveza, velocidade e total compatibilidade:
* **HTML5:** Estruturação semântica de todo o conteúdo e acessibilidade.
* **CSS3:** Design moderno, responsivo (adaptável a telemóveis, tablets e computadores) e transições suaves.
* **JavaScript (Vanilla):** Responsável por toda a lógica de acessibilidade e pelos efeitos interativos.

## ♿ Recursos de Acessibilidade (Inclusão Digital)

Pensando no alcance e na inclusão de todos os utilizadores, o site conta com um painel de acessibilidade (ícone de engrenagem `⚙️`) que oferece:
* **Alteração de Tema:** Alternância entre o Modo Dia (claro) e o Modo Noite (escuro) para maior conforto visual.
* **Ajuste de Fonte:** Opção de aumentar ou diminuir o tamanho dos textos para facilitar a leitura.
* **Leitura de Conteúdo:** Integração com a API de síntese de voz para narrar os textos da página a utilizadores com deficiência visual.
* **Fechamento Inteligente:** O menu fecha-se automaticamente ao clicar em qualquer área externa do ecrã, melhorando a experiência de navegação.

## 🍃 Easter Egg (Segredo Oculto)

Para trazer um diferencial artístico, dinâmico e surpreendente ao projeto, foi desenvolvido um "Easter Egg" interativo. Ao clicar no emoji da plantinha (`🌱`) no menu superior, o utilizador ativa uma experiência visual única:
* Um efeito de folhas caindo é acionado, gerando uma chuva densa e constante de 120 folhas que caem de forma suave pelo ecrã.
* As folhas são **100% responsivas e escaláveis**, mudando de tamanho automaticamente de acordo com o dispositivo (celular, tablet ou PC).
* O ecrã ganha um filtro verde intenso perfeitamente sincronizado com a mensagem inspiradora central, durando exatamente 5 segundos antes de fazer o reset de forma limpa.
